DuKQa

As the Lead Developer, I designed and built DuKQa, an e-commerce platform for KQ Cargo to facilitate international shipping using Kenya Airways' cargo planes. The platform integrates payment gateways (Mastercard, M-Pesa, Stripe) and provides real-time tracking for exporters and importers.

Key Features

  • Multi-role system (Exporters, Importers, Admin).
  • Payment integrations with Mastercard, M-Pesa, and Stripe.
  • Real-time shipment tracking using KQ Cargo's logistics API.
  • Multi-currency support for international transactions.
  • Email and SMS notifications for order updates.

Technologies & Tools

  • Frontend: Next.js, React, Tailwind CSS
  • Backend: .NET, Entity Framework, RESTful APIs
  • Database: SQL Server, Redis (caching)
  • Integrations: Mastercard, M-Pesa, Stripe, KQ Cargo API
  • DevOps: AWS EC2, RDS, S3, Azure DevOps, Docker

Team & Role

  • Team Size: 3 developers
  • My Role: Lead Developer (Frontend, Backend, DevOps)

Impact & Results

  • Streamlined international shipping for KQ Cargo.
  • Improved user experience with seamless payments and tracking.
  • Scalable platform capable of handling high traffic.
