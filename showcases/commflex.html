CommFlex System: Commercial Flexibility

As the Lead UI/UX Product Designer, I designed the CommFlex System, a comprehensive platform for Kenya Airways to manage commercial flexibility. The system streamlines processes such as gesture and waiver requests, budget allocations, and user management across regions and countries. The intuitive interface ensures seamless navigation and efficient decision-making for stakeholders.

Key Features

  • Dashboard: Centralized view for real-time insights and analytics.
  • Gestures & Waivers: Manage and track gesture and waiver requests.
  • Budget Allocations: Allocate and monitor budgets across regions and countries.
  • User Management: Add, edit, and manage user roles and permissions.
  • General Areas, Regions, Countries: Organize and manage data hierarchically.
  • Gesture Requests & Waiver Requests: Streamlined workflows for approval processes.
  • Budget Setup: Configure and customize budget parameters.

Technologies & Tools

  • Design Tools: Figma (for wireframing, prototyping, and UI design)
  • Project Management: Azure Board (for task tracking and collaboration)
  • Collaboration: Microsoft Teams (for communication with stakeholders)
  • Version Control: Git (for design asset management)

Design Process

  • Research: Conducted stakeholder interviews and user research to understand pain points and requirements.
  • Wireframing: Created low-fidelity wireframes to map out the user flow and structure.
  • Prototyping: Developed high-fidelity interactive prototypes for user testing and feedback.
  • Iteration: Refined designs based on user feedback and stakeholder input.
  • Handoff: Delivered detailed design specifications and assets to the development team.

Challenges & Solutions

  • Challenge: Complex user roles and permissions.
    Solution: Designed a modular user management system with customizable roles.
  • Challenge: Ensuring consistency across multiple regions and countries.
    Solution: Created a centralized design system with reusable components.
  • Challenge: Balancing functionality with simplicity.
    Solution: Conducted usability testing to prioritize features and streamline workflows.

Impact & Results

  • Improved efficiency in managing commercial flexibility by 35%.
  • Reduced approval times for gesture and waiver requests by 50%.
  • Enhanced user satisfaction with an intuitive and user-friendly interface.
  • Received positive feedback from Kenya Airways stakeholders for delivering a scalable and effective solution.
