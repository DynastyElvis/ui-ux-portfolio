Website for Promo Gurus Limited

Promo Gurus Limited is a leading marketing and promotional agency. The website was designed to showcase their services, portfolio, and client success stories. Built with **ReactJS**, **jQuery**, and **Material UI**, the platform offers a modern and responsive user experience.

Key Features

  • Modern and responsive design.
  • Showcase of services and portfolio.
  • Client success stories and testimonials.
  • Contact form for inquiries.
  • Integration with social media platforms.

Technologies & Tools

  • Frontend: ReactJS, jQuery, Material UI
  • Styling: CSS3, Bootstrap
  • Version Control: Git/GitHub
  • Project Management: Trello

Challenges & Solutions

  • Challenge: Ensuring a responsive design for all devices.
    Solution: Used Material UI and Bootstrap for responsive layouts.
  • Challenge: Integrating social media feeds.
    Solution: Used jQuery plugins for seamless integration.
  • Challenge: Optimizing performance for fast load times.
    Solution: Implemented lazy loading and image optimization.

Impact & Results

  • Improved user engagement with a modern and intuitive design.
  • Increased client inquiries through the contact form.
  • Enhanced brand visibility with social media integration.
