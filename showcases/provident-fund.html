Provident Fund Website

The Provident Fund Website is a comprehensive platform designed for pension and group members of the Kenya Airways Group, including Kenya Airways, Jambojet, Precision Air, and Fahari Aviation. The system provides seamless access to pension information, member dashboards, and resource centers. Built with **Next.js** and integrated with **Payload CMS** for continuous updates, the platform ensures scalability and reliability.

Key Features

  • Member login and dashboard integration.
  • About Us, Downloads, Resource Center, Investments, Contacts, and Careers pages.
  • Continuous updates via Payload CMS.
  • Deployed on AWS EC2 with Azure DevOps pipelines.
  • Used by 6 companies under the Kenya Airways Group.

Technologies & Tools

  • Frontend: Next.js, Tailwind CSS
  • Backend: Payload CMS, RESTful APIs
  • DevOps: Azure DevOps, AWS EC2
  • UI/UX Design: Figma
  • Project Management: Azure Boards

Challenges & Solutions

  • Challenge: Integrating Payload CMS for continuous updates.
    Solution: Built custom APIs for seamless CMS integration.
  • Challenge: Ensuring secure member login and data access.
    Solution: Implemented role-based access control and encryption.
  • Challenge: Scaling for multiple companies under the Kenya Airways Group.
    Solution: Used AWS EC2 with auto-scaling and load balancing.

Impact & Results

  • Streamlined pension management for Kenya Airways Group members.
  • Improved user experience with a modern and intuitive interface.
  • Enhanced data security and scalability.
  • Successfully deployed and used by 6 companies.
