iRecruitment System: Revolutionizing Recruitment for Kenya Airways

The iRecruitment System is a comprehensive recruitment platform designed for Kenya Airways to streamline and modernize their hiring processes. The system integrates seamlessly with Kenya Airways' main website and provides tailored workflows for various user roles, including Admin, HR, HR Manager, HR Partner, Divisional Manager, CEO, Chief HR, Line Manager, and Panelists. Key features include role-specific dashboards, interview scheduling, assessment management, and real-time updates powered by Keycloak for secure authentication and authorization.

Key Features

  • Recruitment website integrated with Kenya Airways' main website.
  • User roles: Admin, HR, HR Manager, HR Partner, Divisional Manager, CEO, Chief HR, Line Manager, Panelists.
  • Custom dashboards and workflows for each role.
  • Calendar for scheduling interviews and assessments.
  • Interview and assessment management.
  • Keycloak for authentication and authorization.

Technologies & Tools

  • Frontend: Next.js, Tailwind CSS
  • Backend: .NET, RESTful APIs
  • Database: SQL Server
  • Authentication: Keycloak
  • DevOps: Azure DevOps, AWS (EC2, RDS, S3)
  • UI/UX Design: Figma
  • Project Management: Azure Boards

Challenges & Solutions

  • Challenge: Complex user roles and permissions.
    Solution: Implemented Keycloak for role-based access control.
  • Challenge: Real-time updates for interview scheduling.
    Solution: Used WebSocket for real-time calendar updates.
  • Challenge: Scaling the system for high user traffic.
    Solution: Deployed on AWS with auto-scaling and load balancing.

Team & Role

  • Team Size: Part of a larger team of developers, designers, and QA engineers.
  • My Role: Software Engineer (Frontend, Backend, DevOps)

Impact & Results

  • Streamlined recruitment processes for Kenya Airways.
  • Improved user experience with role-specific dashboards.
  • Scalable system capable of handling high user traffic.
  • Enhanced security with Keycloak authentication.
